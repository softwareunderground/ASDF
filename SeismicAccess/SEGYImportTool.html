SEG-Y reading tool

We have no chance without a tool that reads one or more SEG-Y files and establishes the parameters so it can bu used by the input adapter. Compare OpendTect's wizard (6.0 and higher).

User should be able to indicate keys (inl/crl, X/Y, tracenr/cdp), alternate headers (like SP), add missing data. The system itself should make sure all harvested header data can be kept available via semantic keys.

The resulting setup and scan files should be sharable.
